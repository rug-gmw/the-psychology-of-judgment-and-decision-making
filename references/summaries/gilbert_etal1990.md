# Unbelieving the Unbelievable: Some Problems in the Rejection of False Information

Gilbert, D. T., Krull, D. S., & Malone, P. S.

__This article challenges Descartes’ view that people first comprehend ideas neutrally before deciding whether to believe them, instead supporting Spinoza’s claim that all information is initially accepted as true and only later rejected if proven false. Through three experiments, the authors demonstrate that interrupting cognitive processing increases the likelihood of mistaking false information for true, suggesting that rejecting falsehoods requires additional mental effort.__

The article begins by contrasting two historical models of belief formation: Descartes’ view that ideas are first represented neutrally and later assessed for truth, and Spinoza’s argument that all information is initially accepted as true and only later "unaccepted" if found false. The authors use a library metaphor to illustrate these models—Descartes’ system tags all books as either true or false, while Spinoza’s system assumes all books are true unless explicitly tagged as false. This distinction becomes critical when cognitive processing is interrupted, as Spinoza’s model predicts that false information will be mistaken for true, whereas Descartes’ model does not anticipate such an asymmetry.

The first experiment tested this prediction by presenting participants with novel linguistic propositions (e.g., "A twyrin is a doctor") labeled as either true or false, with some trials interrupted by a secondary task. Results confirmed Spinoza’s hypothesis: interruptions impaired participants’ ability to correctly identify false propositions as false but had no effect on true propositions. This asymmetry persisted even when participants were explicitly warned about the veracity of the information before processing it, suggesting that initial acceptance of information as true is not easily overridden by prior knowledge.

The second experiment extended these findings to non-linguistic stimuli, using facial expressions of emotion. Participants viewed smiling faces labeled as either genuinely happy or deceptive, with some trials interrupted. Again, interruptions led participants to misidentify deceptive smiles as genuine, reinforcing the idea that even iconic information is initially accepted as true before being reassessed.

The third experiment examined whether merely comprehending a proposition—without assessing its truth—affects later judgments. Participants learned facts about an imaginary animal ("glarks") and later assessed propositions about them, some of which they had previously read without evaluating. Results showed that prior comprehension increased the likelihood of later judging false propositions as true, suggesting that the mental representation of merely comprehended information resembles that of believed information.

The discussion situates these findings within broader psychological research, noting parallels in psycholinguistics (e.g., faster verification of true statements), social psychology (e.g., increased gullibility under cognitive load), and hypothesis testing (e.g., confirmatory bias). The authors argue that the Spinozan model aligns with the automaticity of perception, where sensory input is initially accepted as true before being critically evaluated. They conclude that the human mind may be inherently predisposed to believe first and doubt later, a tendency that is both efficient and prone to error. This challenges the Cartesian intuition that belief is a controlled, deliberative process, instead framing it as an automatic and often uncontrollable byproduct of comprehension.
